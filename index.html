Core Features

Everything you need to build together, fast.

Production-grade infrastructure with developer-first ergonomics.

⚡

Real-Time Sync

Socket.IO-powered WebSocket connections with optimistic updates. Changes propagate across all clients in under 50ms with automatic conflict resolution.

👥

Live Presence

See who's viewing each board in real-time. Redis-backed user presence tracking with avatar display and active cursor positions.

🔐

Auth & Access Control

JWT tokens with refresh token rotation. Role-based permissions at the workspace and board level — owner, editor, and viewer roles.

📋

Kanban Boards

Drag-and-drop card management across unlimited columns. Full CRUD for boards, columns, and cards with instant sync to all collaborators.

💬

Board-Level Chat

Scoped real-time text chat within each board. Keeps communication contextual and eliminates context-switching to external tools.

🏗️

Turborepo Monorepo

Web and server apps share TypeScript types and Zod schemas through a shared package. Parallel builds, incremental caching, zero drift.

Developer Experience

Up and running in 4 commands.

Docker Compose handles Postgres and Redis. Turborepo runs everything in parallel with shared TypeScript types across the monorepo.

terminal
# Install all workspace dependencies
npm install
 
# Start Postgres + Redis via Docker
npm run docker:up
 
# Run DB migrations
npm run db:migrate
 
# Start frontend + backend (parallel)
npm run dev
 
# ✓ Frontend → http://localhost:3000
# ✓ Backend → http://localhost:4000
